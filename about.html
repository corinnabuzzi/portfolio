About

I come from a humanities background, studying French and German and literature. I fell in love with the mechanics of it: grammar, structure, the logic underneath. I went to university to pursue linguistics, but quickly realized the academic world didn't work for me, and I left.

I was already programming as a hobby: building things with Pygame, tinkering with HTML since I was a kid, but it never felt like a possible career. Once I did start taking it into consideration, though, a lot clicked: I found out the same instinct I had for parsing language applied directly to parsing systems. Syntax is syntax. Structure is structure.

I've been self-taught my whole life: I created my first blog at eight, picked up guitar at 13, wrote my first lines of Python at 15. I don't think that's going to stop any time soon. I come home, and I study. I build things. I always have and, hopefully, always will.

Right now I'm focused on Java backend and JavaScript frontend development, with Node and Angular. I'm passionate about Linux and its philosophy, and plan to start contributing to open source projects and tools.

If I ever get the chance to work in digital humanities (computational linguistics or anything at that intersection) I'd take it without thinking twice. But even if I don't, I'll keep studying language and computation on my own. That's not a backup plan, it's the default.
